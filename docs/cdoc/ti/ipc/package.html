DETAILS
This is a package that serves as a container for common header files for various IPC modules supplied with the IPC product.
Documentation for all runtime APIs, instance configuration parameters, error codes macros and type definitions available to the application integrator can be found in the Doxygen documentation for the IPC product. However, the documentation presented in RTSC cdoc should be referred to for information specific to the RTSC module, such as module configuration, Errors, and Asserts.
The following table shows a list of all IPC modules that have common header files. Follow the corresponding links for doxygen or cdoc documentation for each of the modules.
GateMP Multiple processor gate that provides local and remote context protection Doxygen cdoc

HeapMemMP Multi-processor variable size buffer heap Doxygen cdoc
HeapMultiBufMP Multiple fixed size buffer heap Doxygen cdoc
